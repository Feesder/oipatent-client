# OiPatent Client Application

Данный проект представляет веб-приложение, предназначенное для оптимизации документирования с помощью AI и работы с патентами. Проект разработан с использованием фреймворка Next.js.

Frontend-приложение использует backend-сервис OiPatent через HTTP-API.

## Установка
1. Клонирование репозиторий: 
```
git clone https://github.com/Feesder/oipatent-client.git
cd oipatent-client
```
2. Запуск приложения
```bash
npm run dev
```
Приложение запускается по адресу http://localhost:3000
